## 開発言語
・Ruby 3.3.0  
・Rails 6.1.7.8  

## 実行手順
$ git clone https://github.com/Shoko-Kikuchi/kaonajimi.app.git  
$ cd kaonajimi_app  
$ bundle install  
$ rails db:create && rails db:migrate  
$ rails s
